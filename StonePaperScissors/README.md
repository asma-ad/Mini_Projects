## Stone Paper Scissors

This is a game played between two players, you and the computer, where each player in this game forms one of three shapes and the winner is decided as per the rules of the game.

Game rules:
Rock vs. Paper -> Paper wins.

Paper vs. Scissor -> Scissor wins.

Scissor vs. Rock -> Rock wins.


================================================================================

                              Project requirements:

================================================================================

* Ask for how many rounds the game will be.
* Start each round: Player vs Computer.
* Show the results with each round.
* If the computer wins the round, ring a bill and make the screen red.
* If the player wins the round show a green screen.
* After all rounds show a game over, then print game results. Then ask the user if they want to play again.
